# How To Be Cyber Super

Educational apps for teaching children cybersecurity concepts and safe online practices.

## Overview

This repository contains interactive educational applications designed to help children (ages 7-11) learn important cybersecurity concepts through engaging, game-like experiences.

## Projects

### Cyber Super App

An interactive React-based web application that teaches kids safe browsing practices through:

- **Training Protocols**: Three core lessons (Default NO, 3 Always, Neverland)
- **App Hero**: Learn about app traps and how to take control of app settings
- **Practice Mode**: 15 interactive scenarios to test knowledge
- **Kid-Friendly UI**: Engaging design with animations and interactive elements

[View Cyber Super App →](./cyber-super-app)

## Tech Stack

- React 19
- TypeScript
- Vite
- TailwindCSS v4

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/HowToBeCyberSuper.git
cd HowToBeCyberSuper
```

2. Navigate to the cyber-super-app directory:
```bash
cd cyber-super-app
```

3. Install dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm run dev
```

5. Open your browser and navigate to `http://localhost:5173`

## Building for Production

```bash
cd cyber-super-app
npm run build
```

The production-ready files will be in the `dist` directory.
